ANSI (terminal-only) game engine written in GTK/C++, using mostly stdio.h (and other standard libraries)

Build options:
- xpm_name_support 0 in ansigame.cpp will save you ~40kb on your executable size

Build and run (GTK/Ubuntu):
`$ ./go`
